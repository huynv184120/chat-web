# chat-web
This is a web application that allows you to create rooms to chat with other people.
# Build with
  - ## Reactjs
  - ## Nodejs
  - ## Socket.io
# Getting Started
## Prerequisites
- npm 
- nodejs
- reactjs
## Installation
1. Clone the repo
```
git clone 
```
2. Install NPM packages
```
npm install
```
## Usage
I have deployed this project in [chat app](https://aqueous-savannah-70736.herokuapp.com/)
